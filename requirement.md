# Medical Educational Platform – Migration & Expansion

## General Objective
Migrate and modernize a medical educational platform currently hosted on Firebase, preserving all existing information (users, questions, progress, and content), and transform it into a scalable, professional, multi-device platform ready for international expansion.

## Current Features to Maintain

### Question Bank
- Study mode
- Exam mode
- Selection by:
  - Specialty
  - Number of questions
  - Exam time
  - Topics/subtopics
- Feedback and answer explanations
- Performance statistics
- User progress tracking
- Exam history for review

### Flashcards
- Review system
- Flashcards by specialty/topic
- Spaced repetition algorithm (ideal)

### Video Classes
- Access to video classes organized by modules
- Visual student progress
- Resume where you left off

### Calendar
- Ability to upload attachments (e.g., PDF study schedules) for students to view

### Subscriptions
- Time-based subscription plans:
  - Monthly
  - Quarterly
  - Annual (ability to show/hide each plan)
- Current Stripe integration
- Auto-renewal
